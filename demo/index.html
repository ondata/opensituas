Caso d'uso sul Comune di Messina

opensituas interroga il SITUAS (ISTAT) — la fonte autoritativa dei codici e delle denominazioni di tutte le unità territoriali italiane, dall'Unità d'Italia a oggi. Questa demo mostra come estrarre e leggere dati reali usando il Comune di Messina come filo conduttore.

Come funziona

Il flusso di lavoro in 3 passi

1 Trova il report

Sfoglia il catalogo dei 74 microservizi SITUAS e trova il pfun giusto.

opensituas catalog "comuni"
opensituas which "popolazione"
2 Capisci le colonne

Leggi descrizione e schema del report prima di scaricare i dati.

opensituas info 74
3 Scarica i dati

Snapshot a una data, intervallo storico o query su una singola unità.

opensituas get 74
opensituas count 61 --date 01/01/2000
opensituas storia comune Messina
Tutti i comandi accettano -o json / -o csv per pipeline con jq o DuckDB. Exit code 2 = errore API (messaggio su stderr); 0 = ok.

Codice ISTAT del Comune cerca-codice
083048
Codice alfanumerico ISTAT
Valido dal 17/03/1861 Ancora attivo

Il codice ISTAT è l'identificativo univoco del comune nel sistema SITUAS. È la chiave di join (PRO_COM_T) per incrociare qualsiasi dataset pubblico italiano: dati elettorali, bilanci comunali, dati di popolazione, open data PA.

Se il nome è ambiguo (più comuni con lo stesso nome), cerca-codice elenca tutte le forme storiche con i rispettivi intervalli di validità.

$ opensituas cerca-codice comune Messina

Storia delle variazioni territoriali storia --dettaglio

Dal 1861 a oggi, Messina ha subito 5 variazioni territoriali documentate con provvedimento normativo. Il comando storia le ricostruisce tutte, con riferimento a leggi e decreti reali.

Prima registrazione nel sistema SITUAS — data di inizio dell'Unità d'Italia.

Aggregata al Comune di Messina parte della frazione Nunzio Bicchieri, staccata dal Comune di Calvaruso (cod. 083802).

17 aprile 1928
AQES Acquisizione per estinzione di Santo Stefano di Briga
Aggregato al Comune di Messina il territorio del soppresso Comune di Santo Stefano di Briga (cod. 083807).
R.D. N. 546 dell'11/03/1928 · G.U. N. 78 del 02/04/1928
2 giugno 1929

Nuovo Comune di Villafranca Tirrena (cod. 083105) costituito con i territori dei soppressi comuni di Bauso, Calvaruso, Saponara Villafranca e con le frazioni Divieto e Serro staccate da Messina.

Nessuna variazione dal 1986. Messina è stabile amministrativamente da 40 anni. Il report pfun 129 (variazioni dal 1991) non riporta eventi per questo comune.

Riferimento rapido

Cosa puoi chiedere a opensituas

La colonna Con la Skill AI mostra come porre la domanda in linguaggio naturale a un agente Claude Code con la skill situas-explorer installata. Installa la skill →

Domanda di esempio Comando opensituas Con la Skill AI
Qual è il codice ISTAT del mio comune? opensituas cerca-codice comune "Messina" "Trova il codice ISTAT di Messina"
Quando è nato? Ha cambiato nome o confini? opensituas storia comune "Messina" --dettaglio "Raccontami la storia del comune di Messina"
Quanti abitanti ha? Quanto è grande? opensituas -o json get 74 | jq '.[] | select(.PRO_COM_T == "083048")' "Popolazione e superficie di Messina"
È un comune di montagna? È sul mare? opensituas -o json get 73 | jq '.[] | select(.PRO_COM_T == "083048")' "Che caratteristiche territoriali ha Messina?"
È un polo di servizi o un'area periferica? opensituas -o json get 75 | jq '.[] | select(.PRO_COM_T == "083048")' "Messina è un'area interna?"
A quale bacino lavorativo appartiene? opensituas -o json get 408 | jq '.[] | select(.PRO_COM_T == "083048")' "A quale SLL appartiene Messina?"
Qual è il suo codice europeo NUTS3? opensituas -o json get 76 | jq '.[] | select(.PRO_COM_T == "083048")' "Qual è il codice NUTS3 di Messina?"
Quanti comuni aveva l'Italia nel 2001? opensituas -o json count 61 --date 01/01/2001 | jq '.[0].NUM_ROWS' "Quanti comuni aveva l'Italia nel 2001?"
Come sono cambiati i codici ISTAT dal 2001 a oggi? opensituas -o csv get 99 --from 21/10/2001 --to 31/12/2024 --out mapping.csv "Mappami i codici comuni dal 2001 a oggi"
Quali comuni si sono fusi dal 2010? opensituas -o json get 129 | jq '[.[] | select(.ANNO >= "2010" and (.DESC_COD_VARIAZIONE | contains("usione")))]' "Elenca le fusioni di comuni dal 2010"
